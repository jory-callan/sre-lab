这个的 SID 是 helowin ，密码 root/helowin helowin/helowin
但是我不希望使用这个，因为无法自定义 ORACLE_SID=ORCL

```bash
docker run -d --name oracle_11g \
-p 1521:1521 \
--restart=always \
-e TZ=Asia/Shanghai \
-v /data/oracle/data/admin:/home/oracle/app/oracle/admin \
-v /data/oracle/data
registry.cn-hangzhou.aliyuncs.com/helowin/oracle_11g 
# 或者使用 docker HUB 上的 akaiot/oracle_11g:latest 

```

此镜像的 oracle 默认参数

```bash
hostname: localhost
port: 1521
sid: helowin
username: system
password: helowin
```
