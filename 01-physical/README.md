# 01-physical - 物理层

## 📂 目录结构

```
01-physical/
├── linux/
│   ├── init/           # Linux 系统初始化脚本
│   └── scripts/        # Linux 工具脚本
├── windows/            # Windows 相关
├── macos/              # macOS 相关
└── network/            # 网络相关文档和配置
```

---

## 🎯 用途

这一层包含：
- 操作系统初始化脚本
- 各类工具脚本
- 网络拓扑文档
- 物理机相关配置

---

## 📖 子目录说明

| 目录 | 说明 |
|------|------|
| [linux/init/](./linux/init/) | Linux 系统初始化（装机后第一步运行） |
| [linux/scripts/](./linux/scripts/) | Linux 常用工具脚本 |
| [windows/](./windows/) | Windows 相关脚本和文档 |
| [macos/](./macos/) | macOS 相关脚本和文档 |
| [network/](./network/) | 网络拓扑、交换机配置等 |
