## и®ҝй—®ж–№ејҸ

| ж–№ејҸ | иҜҙжҳҺ |
|------|------|
| `redis-redis.<ns>` | дё»иҠӮзӮ№ ServiceпјҲиҜ»еҶҷпјү |
| `redis-redis-<idx>.redis-redis-headless.<ns>` | еҗ„ Pod зӣҙиҝһ |
| `redis-redis-sentinel.<ns>` | Sentinel Service |

## еӯҳеӮЁ

- **Redis data:** 5Gi PVCпјҢй»ҳи®Ө StorageClass
- **Sentinel data:** 1Gi PVCпјҢй»ҳи®Ө StorageClass

## жіЁж„ҸдәӢйЎ№

1. **KubeBlocks з®ЎзҗҶ** вҖ” иҜ·еӢҝжүӢеҠЁдҝ®ж”№ Pod/ServiceпјҢйҖҡиҝҮ Cluster CR еҸҳжӣҙ
2. **ж•…йҡңеҲҮжҚў** вҖ” Sentinel иҮӘеҠЁеӨ„зҗҶпјҢKubeBlocks дјҡжӣҙж–° Service и·Ҝз”ұ
3. **еҜҶз Ғ** вҖ” й»ҳи®Өж— еҜҶз ҒпјҢеҰӮйңҖи®ӨиҜҒеҸҜйҖҡиҝҮ KubeBlocks й…ҚзҪ®
4. **жҢҒд№…еҢ–** вҖ” RDB/AOF ж–Үд»¶дҝқеӯҳеңЁ PVC дёӯ
