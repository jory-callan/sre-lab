# 部署 — Kite 测试实例

## 前置依赖

- ingress-nginx
- cert-manager + internal-ca ClusterIssuer
- nfs-client StorageClass（用于 SQLite 持久化）

## 部署

```bash
bash install.sh install
```

## 卸载

保留数据（仅移除 Helm release，PVC 不动）:
```bash
bash install.sh uninstall
```

完全清理（删除命名空间及所有资源）:
```bash
bash install.sh purge
```
