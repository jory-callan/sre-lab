# 交付 — kdebug 测试实例

## 访问方式

| 方式 | 地址 | 用途 |
|------|------|------|
| HTTPS | https://kdebug.czw-sre.internal | 集群内 HTTP 调试 |
| NodePort | `<任一节点IP>:30302` | 集群外调试 |
| ClusterIP | kdebug.kdebug.svc.cluster.local:80 | 集群内服务调用 |

## API

| 路径 | 方法 | 说明 |
|------|------|------|
| /ping | GET | 健康检查，返回 `pong` |
| / | GET/POST | Echo 回显请求信息 |

## 注意事项

- 无认证，仅限内部网络使用
- 默认 2 副本，HPA CPU >70% 自动扩容到 5 副本
- PDB 保证至少 1 副本可用
