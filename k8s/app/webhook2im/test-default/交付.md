# 交付 — webhook2im 测试实例

## 访问方式

| 方式 | 地址 | 用途 |
|------|------|------|
| HTTPS | https://webhook2im.czw-sre.internal | Webhook 接收端 |

## 发送测试 Webhook

```bash
curl -X POST https://webhook2im.czw-sre.internal/webhook \
  -H "Content-Type: application/json" \
  -d '{"text":"hello world"}'
```

## 配置说明

应用通过环境变量配置，可在 `values.yaml` 的 `env` 中自定义。

## 注意事项

- 无认证，仅限内部网络使用
- 默认 1 副本，如需 HA 请调整 replicaCount 并启用 PDB
