# 部署 — Gitea 测试实例

## 前置依赖

- ingress-nginx
- nfs-client StorageClass（用于 NFS RWX PVC）

## 部署

```bash
bash install.sh install
```

## 卸载

保留数据（仅移除 Helm release，PVC 不动）:
```bash
bash install.sh uninstall
```

完全清理（删除命名空间及所有资源）:
```bash
bash install.sh purge
```

## 升级

```bash
# 修改 test-default/values.yaml 中 image.tag
bash install.sh install
```
