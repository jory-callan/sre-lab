# 交付 — Gitea Actions Runner

## 架构

```
Gitea (gitea-http.gitea:3000)
    │ 注册 → 轮询任务
    ▼
act_runner (gitea-runner)
    │ host 网络模式
    ├── docker build/push → Nexus 5001
    ├── kubectl apply → K3s API
    └── git clone → Gitea NodePort
```

## 存储

- PVC: 1Gi, nfs-client
- 存储 `.runner` 注册文件，Pod 重启免重新注册

## ci-deployer

集群级 ServiceAccount，用于 CI/CD 流水线执行 kubectl apply。token 存到 Gitea Secret `KUBE_TOKEN`。
