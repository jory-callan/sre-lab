# 部署 — Gitea Actions Runner

## 前置依赖

- Gitea 已部署
- runner-base Docker 镜像已构建并推送到 Nexus（`bash runner-image/build-runner-image.sh`）

## 部署

```bash
# 1. 配置 registrationToken
vim chart/values.yaml

# 2. 部署
bash install.sh install
```

## 卸载

保留 PVC（.runner 注册文件）:
```bash
bash install.sh uninstall
```

完全清理:
```bash
bash install.sh purge
```
