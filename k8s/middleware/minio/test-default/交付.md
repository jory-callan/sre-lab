# 交付 — MinIO Tenant

## 架构

```
用户 → https://minio-api.czw-sre.internal (S3 API)
        https://minio.czw-sre.internal (Console)
          │
          ▼
    ingress-nginx (cert-manager TLS)
          │
          ▼
    minio Service (ClusterIP:80) → Tenant Pod
          │
          └── 4× PVC (nfs-client, 2Gi/卷)
```

## 预置 Bucket

| Bucket | 用途 |
|--------|------|
| velero | Velero 集群备份 |
| vm-metrics | VictoriaMetrics 长期指标 |
| vm-logs | VictoriaMetrics Logs |
| public | 公开数据 |
| private | 内部数据 |
| postgres-backup | 数据库备份 |

## 服务账号

| 账号 | 权限 |
|------|------|
| svc-poweruser | 所有 bucket 读写 |
| svc-private | 仅 private bucket 读写 |

## 版本锁定

当前版本组合是 MinIO 最后一个完整 WebUI 版本，不再主动升级。

| 组件 | 版本 | 原因 |
|------|------|------|
| Operator | v5.0.18 | v6.x 移除 Console CRD，v5.x 最后一个版本 |
| MinIO 镜像 | RELEASE.2025-04-22 | 最后一个完整的 Web Console |
