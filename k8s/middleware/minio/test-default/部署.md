# 部署 — MinIO Tenant

## 前置依赖

- nfs-client StorageClass
- cert-manager + internal-ca ClusterIssuer
- ingress-nginx

## 部署

```bash
bash install.sh install
```

## 卸载

保留数据（仅删除 Tenant CR，PVC 保留）：
```bash
bash install.sh uninstall
```

完全清理：
```bash
bash install.sh purge
```
