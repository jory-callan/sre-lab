# 部署说明

## 前置条件
- CloudNativePG Operator 已安装（`../operator/install.sh`）

## 部署步骤
1. 配置 MinIO 依赖：`bash dep-minio-setup.sh`
2. 创建 Secret：`kubectl apply -f cr-secret.yaml`
3. 创建实例：`kubectl apply -f cr-cluster.yaml`
4. 配置 Service：`kubectl apply -f cr-service-external.yaml`
5. 配置监控：`kubectl apply -f cr-pod-monitor.yaml`

> 可直接使用 `bash install.sh install` 一键完成
