## 部署管理

### 1. 部署 Temporal

```bash
# 部署单体版
kubectl apply -f temporal-simple.yaml

# 部署 MySQL
kubectl apply -f temporal-simple-mysql.yaml
```

### 2. 更新部署

```bash
# 更新配置并重启 Server
kubectl apply -f temporal-simple.yaml
kubectl rollout restart deployment/temporal-simple-server -n temporal-simple

# 查看部署状态
kubectl rollout status deployment/temporal-simple-server -n temporal-simple
```

### 3. 删除部署

```bash
# 谨慎！删除整个命名空间
kubectl delete namespace temporal-simple
```

## Pod 管理

### 1. 查看 Pod

```bash
# 查看所有 Pod
kubectl get pods -n temporal-simple

# 查看 Pod 详情
kubectl describe pod <pod-name> -n temporal-simple

# 实时查看 Pod 变化
kubectl get pods -n temporal-simple -w
```

### 2. 进入 Pod

```bash
# 进入 admin-tools
kubectl exec -it -n temporal-simple deploy/temporal-simple-admintools -- sh

# 进入 Server
kubectl exec -it -n temporal-simple deploy/temporal-simple-server -- sh
```

## Namespace 操作

### 1. 列出 Namespaces

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal operator namespace list --address temporal-simple-frontend:7233
```

### 2. 查看 Namespace 详情

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal operator namespace describe default --address temporal-simple-frontend:7233
```

### 3. 创建 Namespace

**推荐用 wrapper 脚本（自动加 Retention 1年）：**
```bash
# 先用 wrapper 脚本
temporal operator namespace create my-namespace --address localhost:7233
```

**或者手动加参数（必须加 `--retention`）：**
```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal operator namespace create my-namespace \
  --address temporal-simple-frontend:7233 \
  --retention 8760h0m0s
```

### 4. 更新 Namespace

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal operator namespace update my-namespace \
  --address temporal-simple-frontend:7233 \
  --retention 8760h0m0s
```

### 5. 删除 Namespace

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal operator namespace delete my-namespace \
  --address temporal-simple-frontend:7233
```

## Workflow 操作

### 1. 列出 Workflows

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal workflow list \
  --address temporal-simple-frontend:7233 \
  --namespace default
```

### 2. 查看 Workflow 详情

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal workflow describe <workflow-id> \
  --address temporal-simple-frontend:7233 \
  --namespace default
```

### 3. 终止 Workflow

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal workflow terminate <workflow-id> \
  --address temporal-simple-frontend:7233 \
  --namespace default
```

### 4. 终止所有 Workflow

```bash
kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal workflow terminate --all \
  --address temporal-simple-frontend:7233 \
  --namespace default
```

## 日志与排查

### 1. 查看 Server 日志

```bash
# 查看最后 50 行
kubectl logs -n temporal-simple deploy/temporal-simple-server --tail=50

# 实时跟踪
kubectl logs -n temporal-simple deploy/temporal-simple-server -f

# 查看前一个实例
kubectl logs -n temporal-simple deploy/temporal-simple-server -p
```

### 2. 过滤日志

```bash
# 只看 ERROR
kubectl logs -n temporal-simple deploy/temporal-simple-server --tail=100 | grep -i error

# 只看 WARN
kubectl logs -n temporal-simple deploy/temporal-simple-server --tail=100 | grep -i warn
```

### 3. 查看事件

```bash
kubectl get events -n temporal-simple --sort-by='.lastTimestamp'
```

### 4. 查看资源

```bash
# 查看所有资源
kubectl get all -n temporal-simple

# 查看 ConfigMap
kubectl get configmap -n temporal-simple

# 查看 Secret
kubectl get secret -n temporal-simple
```

## 端口转发

### 1. Web UI

```bash
# 终端 1
kubectl port-forward -n temporal-simple svc/temporal-simple-web 8080:8080
```
访问：`http://localhost:8080`

### 2. Frontend

```bash
# 终端 2
kubectl port-forward -n temporal-simple svc/temporal-simple-frontend 7233:7233
```
连接：`localhost:7233`

## 工具脚本使用

### 1. temporal-tool.sh

```bash
# 创建 Namespace（自动 1年 Retention）
./temporal-tool.sh create-ns my-namespace

# 列出 Namespaces
./temporal-tool.sh list-ns

# 查看 Namespace
./temporal-tool.sh describe-ns my-namespace

# 更新 Retention
./temporal-tool.sh update-retention my-namespace

# 查看状态
./temporal-tool.sh status

# 查看日志
./temporal-tool.sh logs

# 进入 Shell
./temporal-tool.sh shell

# 快速验证
./temporal-tool.sh verify

# 端口转发
./temporal-tool.sh port-forward
```

## 快速检查清单

部署后运行：

```bash
# 1. 查看 Pod
kubectl get pods -n temporal-simple

# 2. 查看 Server 日志（无 ERROR）
kubectl logs -n temporal-simple deploy/temporal-simple-server --tail=50

kubectl exec -n temporal-simple deploy/temporal-simple-admintools -- \
  temporal operator namespace describe default --address temporal-simple-frontend:7233

# 4. 查看 Jobs
kubectl get jobs -n temporal-simple
```
