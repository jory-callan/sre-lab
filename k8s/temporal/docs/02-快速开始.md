## 快速部署

### 1. 部署单体版（推荐开发用）

```bash
# 创建命名空间
kubectl create namespace temporal-simple

# 部署 MySQL
kubectl apply -f temporal-simple-mysql.yaml

# 部署 Temporal
kubectl apply -f temporal-simple.yaml

# 等待部署完成
kubectl wait --for=condition=ready pod -n temporal-simple --timeout=300s
```

---

### 2. 访问 Web UI

```bash
# 在终端 1：转发 Web UI
kubectl port-forward -n temporal-simple svc/temporal-simple-web 8080:8080

# 在终端 2：转发 Frontend
kubectl port-forward -n temporal-simple svc/temporal-simple-frontend 7233:7233
```

然后在浏览器打开：`http://localhost:8080`

### 4. 在本地用 Wrapper 脚本（推荐）

```bash
# 1. 下载脚本
scp root@你的服务器:/root/temporal-deploy/temporal-wrapper.sh ~/bin/temporal

# 2. 给执行权限
chmod +x ~/bin/temporal

# 3. 确保 ~/bin 在 PATH 最前面
echo 'export PATH="$HOME/bin:$PATH"' >> ~/.zshrc  # 或者 ~/.bashrc

# 4. 重新加载 shell 配置
source ~/.zshrc
```

**测试一下：**
```bash
temporal operator namespace create test-auto --address localhost:7233
```

你会看到 wrapper 自动添加 `--retention 8760h0m0s！
