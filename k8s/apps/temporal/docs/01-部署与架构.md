## 架构介绍

### 1. 单体版架构 (`temporal-simple`)

**4 个服务：**
```
temporal-simple-mysql       - MySQL 数据库
temporal-simple-server      - 单体 Temporal Server (包含 frontend+history+matching+worker)
temporal-simple-web         - Web UI
temporal-simple-admintools  - 管理工具
```

**优势：**
- ✅ 资源占用少
- ✅ 部署简单
- ✅ 适合开发和测试

### 2. 分布式版架构 (`temporal-test`)

**7 个服务：**
```
temporal-mysql              - MySQL 数据库
temporal-frontend           - Frontend
temporal-history            - History
temporal-matching           - Matching
temporal-worker             - Worker
temporal-web                - Web UI
temporal-admintools         - 管理工具
```

**优势：**
- ✅ 高可用
- ✅ 可扩展
- ✅ 适合生产环境
