## 5. 风险项

### 🟡 中风险：local-path 磁盘空间未知

未检查宿主机磁盘使用率。如果根分区满了，所有 PVC 都会受影响。
**建议**: 登录任一节点执行 `df -h` 检查 `/var/lib/rancher/k3s/storage/` 所在分区。

### 🟢 低风险：单副本服务

以下服务为单副本，重启时有短暂中断：
- kite (1 replica — SQLite 限制)
- Grafana (1 replica)
- Prometheus (1 replica)
- redis-standalone (1 replica)

Redis replication/sentinel 为 3 副本无此风险。

### 🟢 低风险：历史事件中存在镜像拉取失败

Redis 7-alpine 拉取因镜像源限流失败过（已修复，改用 k3s 全局镜像源 + 7.0.15 稳定版本）。

## 6. 建议

1. **监控宿主机磁盘** — local-path 依赖宿主机磁盘空间，建议定期检查 `df -h`

3. **修改 Grafana admin 密码** — `kubectl edit secret -n monitoring kube-prometheus-stack-grafana`
